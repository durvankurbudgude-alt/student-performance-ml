# Student Performance Prediction (ML Project)

## Overview
This project predicts student performance using machine learning techniques.

## Dataset
- 10,000 rows
- Features include study hours, previous scores, sleep hours, etc.

## Models Used
- Linear Regression
- Decision Tree
- Random Forest

## Final Model
Random Forest performed best with:
- R² Score ≈ 0.987
- Cross-validation score ≈ 0.9866

## Key Insights
- Previous Scores is the most important feature (~85%)
- Model shows strong generalization and stability

## Tech Stack
- Python
- Pandas
- Scikit-learn
